

Esse repositório foi criado, primeiramente, com o intuito de desenvolver algumas habilidades relacionadas ao desenvolvimento web usando a plataforma .NET, especificamente a linguagem C#. Dessa forma, criei uma web Api, usando a estrutura padrão (Uso de controllers). Além disso, utilizei o banco de dados Oracle para registrar as operações (CRUD). Ademais, utilizei o Entity Framework (ORM) para otimizar as migrações e execuções no banco de dados.
